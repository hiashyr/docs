Руководство по созданию русского Django веб-приложения
=============================================

Добро пожаловать в подробное руководство по созданию веб-приложения на Django! 
Это пошаговая инструкция, которая проведет вас через весь процесс разработки.

Быстрый старт
-------------

Если вы только начинаете, рекомендуем следовать этому порядку:

1. :doc:`setup` - Настройка окружения и установка Django
2. :doc:`project-structure` - Создание структуры проекта 
3. :doc:`models` - Создание моделей базы данных
4. :doc:`views` - Написание представлений (views)
5. :doc:`templates` - Создание шаблонов
6. :doc:`urls` - Настройка маршрутизации (URLs)
7. :doc:`forms` - Работа с формами
8. :doc:`admin` - Настройка админ-панели
9. :doc:`git` - Работа с гитом
