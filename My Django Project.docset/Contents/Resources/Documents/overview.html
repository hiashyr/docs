Обзор проекта

Мое Django веб-приложение для управления задачами.

Основные функции

  • Управление пользователями - регистрация, аутентификация, профили

  • Система задач - создание, назначение, отслеживание задач

  • REST API - для мобильных приложений и интеграций

  • Админ-панель - управление контентом

Технологии

  • Backend: Django, Django REST Framework

  • База данных: PostgreSQL

  • Фронтенд: HTML/CSS/JavaScript

  • Деплой: Docker, Nginx
