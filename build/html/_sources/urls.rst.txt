Настройка URL-маршрутов
========================

В этом разделе мы настроим систему маршрутизации для нашего Django приложения. URL-конфигурация определяет, какие представления будут обрабатывать запросы по определенным адресам.

Первым делом нужно импортировать функции из views.py, т.к в самих url нужно указывать именно их. Подробнее о views написано тут: :doc:`views`
Сам импорт выполняется одной строкой:

.. code-block:: python:

  from main import views

Теперь используя функции из views мы можем задать ссылки на страницы. В моем случае используется три страницы. Выглядит это следующим образом:

.. code-block:: django:

  from django.contrib import admin
  from django.urls import path
  from main import views

  urlpatterns = [
      path('admin/', admin.site.urls),
      path('', views.index, name='home'),
      path('about', views.about, name='about')
  ]

В данном примере мы указываем для path три параметра: путь (Для главной страницы не ставится /), функция страницы из views и name. Name указывается, чтобы его можно было использовать в html-страницах
